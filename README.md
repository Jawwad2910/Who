# Meridian Post

Independent global news website covering world politics, geopolitics, markets and technology.

**Founded & Edited by J. Asghar**

## Pages
- `index.html` — Homepage with live news
- `geopolitics.html` — Geopolitics section
- `world-politics.html` — World Politics section
- `markets.html` — Markets section
- `technology.html` — Technology section
- `news-article.html` — Live news article page
- `article.html` — Published article page

## Tech
- Live news via RSS (BBC, Al Jazeera, Reuters)
- Articles stored in Supabase
- Pure HTML, CSS, JavaScript — no frameworks
- Hosted on GitHub Pages
